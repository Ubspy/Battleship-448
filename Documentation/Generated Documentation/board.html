Class: board

board()

new board()

Creates a ship object containing 9 rows, 10 columns, a 2d array to act as our board, and an array to contain all of our ships that we put on the board The 2D array is initilized to all 0s. When a miss is detected it is a 1, and the ships are added as ship objects.
